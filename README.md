# Kahoot Bot Flooder

Web-based Kahoot bot flooding tool with real-time terminal output.

## Deploy to Render

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy)

## Features

- Password-protected web panel
- Real-time bot management
- Dark red themed interface
- Mobile responsive
- Socket.io for live updates

## Environment Variables

- `PORT` - Server port (default: 3500)
- `PASSWORD` - Panel access password (default: 2010)
